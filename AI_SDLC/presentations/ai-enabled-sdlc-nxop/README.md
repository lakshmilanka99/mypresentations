# AI-Enabled SDLC for NXOP Presentation

## Overview
This presentation addresses the critical challenge of scaling mainframe modernization for the NXOP program at American Airlines.

## Key Message
The current development cycle time (18 days per feature) is not scalable for a multi-year program involving 200+ legacy applications. AI-enabled SDLC can reduce cycle time by 78% and make the program viable.
